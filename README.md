# Mech Wars

Turned based game:
- Create your mech with different weapons and parts.
- Open creates to get new parts.
- Fight in eventful fights against other mechs.
- Reach the top.


To make game:
- HTML, CSS, JavaScript, JSON, Flask(Python), mySQL
